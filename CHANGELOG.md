- **v3.5** - Quality-of-life: cancellation/checkpoints for long GUI jobs, job-details diagnostics dialog, persistent genre-lookup cache, faster batch-oriented library scans, collaboration-aware artist-style clustering
- **v3.4** - DJ Bridge Mode for multi-seed playlists with genre-aware bridging, union pooling, and comprehensive diagnostics
- **v3.3** - Seed List mode, Sonic/Genre modes, refactored pipeline, blacklist support
- **v3.2** - Windows GUI improvements, MBID enrichment, artist normalization, matching performance
